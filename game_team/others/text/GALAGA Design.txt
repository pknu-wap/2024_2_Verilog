- 게임의 구성
	- 하나의 스테이지, 하나의 목숨
		- 스테이지 확장은 1스테이지 구현 후 고려
	- 모든 스테이지를 클리어하면 게임 승리
		- 스테이지의 모든 적을 제거하면 스테이지 클리어
	- 점수는 곧 승리하는데 걸린 시간
		- 더 작은 점수를 기록한 사람이 더 뛰어난 실력을 가짐

- tick
	- 1-tick == 800K-clk == 16ms

- 필드
	- 640px * 480px
	- enemyZone
		- 640px * 240px
		- 필드의 상단
		- 스테이지가 시작될 때 적이 생성되는 필드
	- emptyZone
		- 640px * 60px
		- 필드의 중앙
	- playerZone
		- 640px * 180px
		- 필드의 하단
		- 스테이지가 시작될 때 플레이어가 생성되는 필드

- 적
	- 36px * 24px
	- 적의 탄과 충돌하지 않음
	- 생성
		- 생성되는 조건
			- 스테이지가 시작됨
		- 생성되는 위치
			- 각 스테이지마다 enemyZone에서 지정된 위치
			- 36px 간격으로 배치됨
	- 이동
		- 속력
			- 1px/tick의 속력으로 이동함
		- 방향
			- 스테이지마다 주어진 패턴으로 이동함
				- 수평 방향으로 이동함
	- 소멸
		- 소멸되는 조건
			- 플레이어의 탄과 충돌함

- 플레이어
	- 24px * 36px
	- 생성
		- 생성되는 조건
			- 스테이지가 시작됨
		- 생성되는 위치
			- playerZone의 중앙
	- 이동
		- 속력
			- 2px/tick의 속력으로 이동함
		- 방향
			- 사용자의 입력에 따라 이동함
				- 수평 방향으로 이동함
	- 소멸
		- 소멸되는 조건
			- 적의 탄과 충돌함

- 적의 탄
	- 4px * 16px
	- 적과 충돌하지 않음
	- 생성
		- 생성되는 조건
			- 적이 탄을 발사함
		- 생성되는 위치
			- 적의 외부, 하단 경계
			- 적과 겹치면 적 위에 표시됨 (== enemy.z-index < enemyBullet.z-index)
	- 이동
		- 속력
			- 2px/tick의 속력으로 이동함
		- 방향
			- 아래로 이동함
	- 소멸
		- 소멸되는 조건
			- 플레이어와 충돌함
			- 플레이어의 탄과 충돌함
			- 화면의 하단 경계를 벗어남

- 플레이어의 탄
	- 4px * 16px
	- 생성
		- 생성되는 조건
			- 플레이어가 탄을 발사함
				- 최대 1회/12tick의 빈도로 발사됨
		- 생성되는 위치
			- 플레이어의 외부, 상단 경계
	- 이동
		- 속력
			- 3px/tick의 속력으로 이동함
		- 방향
			- 위로 이동함
	- 소멸
		- 소멸되는 조건
			- 적과 충돌함
			- 적의 탄과 충돌함
			- 화면의 상단 경계를 벗어남

- Stage 1
	- 시작 조건
		- 게임이 시작됨
	- 종료 조건
		- 스테이지 클리어
			- 모든 적이 소멸됨
		- 게임 패배
			- 플레이어가 소멸됨
	- Phase
		- 15개의 적이 존재
		- 3행 5열 직사각형 배치
		- 모든 적이 소멸할 때까지 Phase 1~4를 반복함
			- Phase 1
				- Step 1
					- 1, 3행의 적이 왼쪽으로 124px 이동함
					- 2행의 적이 오른쪽으로 124px 이동함
				- Step 2
					- 모든 적이 탄을 발사함
			- Phase 2
				- Step 1
					- 모든 적이 원위치로 이동함
				- Step 2
					- 모든 적이 탄을 발사함
			- Phase 3
				- Step 1
					- 1, 3행의 적이 오른쪽으로 124px 이동함
					- 2행의 적이 왼쪽으로 124px 이동함
				- Step 2
					- 모든 적이 탄을 발사함
			- Phase 4
				- Step 1
					- 모든 적이 원위치로 이동함
				- Step 2
					- 모든 적이 탄을 발사함
